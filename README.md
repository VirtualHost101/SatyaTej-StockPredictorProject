# MarketMove-Predictor  
*Stock Price Movement Prediction using Machine Learning (NYSE 2010–2016)*  

---

## Overview
**MarketMove-Predictor** is an end-to-end machine learning project that predicts **next-day stock price movement (Up / Down)** based on historical market data from the **New York Stock Exchange (NYSE)**.  
The project demonstrates the complete data science workflow — from **data preprocessing and feature engineering** to **model training, evaluation, and visualization**.

This project was developed as part of a university machine learning course, with a focus on **financial data modeling, feature design, and predictive performance benchmarking**.
